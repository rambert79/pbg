# 📄 제품 요구사항 정의서 (PRD) - BibleCardMemory

## 1. 프로젝트 개요 (Project Overview)
* **제품명:** BibleCardMemory (성경인물 카드 뒤집기 게임)
* **대상 사용자:** 주일학교 학생 및 교사 (대형 스크린/프리젠테이션 환경에서 단체 진행)
* **핵심 가치:** 성경 속 주요 인물들의 명칭과 특징을 게임(메모리 게임)을 통해 쉽고 재미있게 학습하며, 팀 간 건전한 경쟁과 협동을 유도함.
* **개발 형태:** 브라우저에서 단일 파일로 즉시 실행 가능한 싱글 페이지 웹 애플리케이션 (Single Page HTML5 Web App)

---

## 2. 제품 스펙 및 주요 기능 (Product Specifications)

### 2.1. 게임 난이도 체계 (Difficulty Levels)
성경 속 인물의 신앙 성장 및 직분을 반영한 명칭으로 구성하며, 난이도별 가로 나열식 카드(섹션) 형태로 메인 화면에 배치함.
* **초급 (🟢 목동 모드):** 총 8장 (4쌍) / 4열 배치
* **중급 (🟡 제자 모드):** 총 16장 (8쌍) / 4×4 그리드 배치
* **고급 (🔴 사도 모드):** 총 24장 (12쌍) / 6×4 그리드 배치

### 2.2. 사용자 흐름 및 화면 로직 (User Flow)
1. **메인 화면 (Main Screen):** 가로 배열된 3개의 난이도 카드 중 하나를 마우스/터치로 선택.
2. **팀명 입력 화면 (Team Input Screen):** 난이도 선택 시 팝업 또는 화면 전환을 통해 A팀(Blue)과 B팀(Red)의 이름을 키보드로 입력받음. (기본값: 믿음 팀 / 소망 팀)
3. **인게임 화면 (In-Game Play):** 1:1 턴제 대결 방식으로 게임 진행.
4. **결과 화면 (Game Over / Victory Event):** 모든 카드가 매칭되면 모달 팝업과 축하 이벤트 애니메이션을 통해 최종 스코어 및 최종 우승팀을 선명하게 노출.

### 2.3. 인게임 세부 규칙 및 UI 컴포넌트 (In-Game Rules & UI)
* **상단 스코어보드 (Top Scoreboard):** 
  * A팀(Blue)과 B팀(Red)의 실시간 누적 점수 시각화.
  * 현재 누구의 차례인지 명확히 식별할 수 있는 턴 표시등(Turn Indicator) 구현.
* **카드 디자인 인터랙션 (Card Design & Interaction):**
  * **뒷면 (숫자면):** 연한 노란색 바탕의 부드러운 스카이 블루 톤 문양이 들어간 디자인으로, 중앙에 아이들이 멀리서도 크게 외칠 수 있도록 **크고 명확한 황금색 입체 숫자** 배치.
  * **앞면 (인물면):** 크림색 종이 질감의 테두리와 상하단 교회 아이콘 프레임을 유지하며, 2.5D 카툰 스타일의 성경 인물 일러스트와 하단 한글 명칭(부제 포함) 노출.
  * **컬러 매칭 시스템:** A팀이 짝을 맞추면 해당 카드의 테두리와 배경이 **파란색(A팀 색상)**으로 고정 점령됨. B팀이 맞추면 **빨간색(B팀 색상)**으로 고정 점령됨.
  * **턴 인계 로직:** 카드가 일치하지 않을 경우, 1.2초간 카드 앞면을 보여준 뒤 다시 뒷면(숫자)으로 뒤집히며 상대 팀에게 턴이 넘어감.

## 3. 데이터 모델 및 인물 라인업 (Data & Content Assets)
최대 24장(12쌍) 플레이를 위해 총 12명의 성경 인물 일러스트 및 텍스트 데이터 셋을 확보함.

| 순번 | 인물명 | 카드 하단 부제 | 일러스트 디자인 콘셉트 요약 |
| :--- | :--- | :--- | :--- |
| 1 | **노아** | 방주 & 약속의 무지개 | 방주 앞에서 동물들과 미소 짓는 소년 |
| 2 | **예수님** | 인류의 구원자 & 사랑 | 양을 품에 안고 온화하게 웃으시는 모습 |
| 3 | **모세** | 출애굽의 리더 | 지팡이를 들고 갈라진 홍해 앞에 선 소년 |
| 4 | **다윗** | 믿음의 물매돌 | 조그만 물매돌을 쥔 당찬 목동 소년 |
| 5 | **다니엘** | 사자굴의 기적 | 사자가 순하게 엎드린 동굴 속 소년 |
| 6 | **요셉** | 꿈과 용서의 총리 | 화려한 채색옷을 입고 이집트 궁전 앞에 선 소년 |
| 7 | **요나** | 물고기 뱃속의 예언자 | 고래 뱃속에서 램프를 들고 앉아 있는 소년 |
| 8 | **삼손** | 나실인 & 천하장사 | 풍성하고 긴 머리카락에 나귀 턱뼈를 든 소년 |
| 9 | **아브라함** | 믿음의 조상 & 많은 민족 | 밤하늘의 수많은 별을 가리키며 웃는 노인 |
| 10 | **에스더** | 이스라엘의 왕비 & 지혜 | 화려한 왕관과 자줏빛 드레스를 입은 왕비 |
| 11 | **루디아** | 빌립보의 성도 & 자줏옷 장사 | 고급스러운 자줏빛 천을 두르고 웃는 여인 |
| 12 | **바울** | 이방인의 사도 & 서신서 작가 | 안경을 벗고 두루마리와 깃펜을 든 지적인 노인 |

## 4. 기술 스택 및 제약 사항 (Technical Requirements)
* **Front-end:** HTML5, CSS3, Vanilla JavaScript (외부 라이브러리/프레임워크 배제)
* **환경 제약:** 별도의 백엔드 서버나 DB 없이 브라우저 단독(로컬 환경 파일 실행)으로 구동되어야 하며, 반응형 레이아웃을 통해 다양한 해상도의 빔프로젝터/TV 스크린을 지원해야 함.
* **셔플 로직:** 카드 배치 시 무작위성을 보장하기 위해 `Fisher-Yates Shuffle` 알고리즘 적용.
